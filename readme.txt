== Description ==

Simple Pregnancy Calculator lets you add a datepicher in the page or in the widget area of your site.<br>

The user can select a date and the calculator provides the expected delivery date.<br>

To add the calculator to a page use [simple_pregnancy_calculator]<br>
To add the calculator in a sidebar use the dashboard's Appearence -> Widgets panel<br>
